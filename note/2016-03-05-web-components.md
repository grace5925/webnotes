# Web Components

## Web Components 是什么？

> Web Components consists of several separate technologies. You can think of Web Components as reusable user interface widgets that are creating using open web technology.

2013年 Google I/O 大会上，Eric Bidelman 的演讲视频 [Web Components: A Tectonic Shift for Web Development](https://www.youtube.com/watch?v=fqULJBBEVQE) 很好的介绍了 Web Components 相关知识。


网络组件由以下四种技术组成（每种技术可以独立使用）：

* `HTML Templates` HTML模板
* `Custom Elements` 自定义元素
* `Shadow DOM` 阴影DOM
* `HTML Imports` HTML导入

## HTML Template



网络组件规范定义了如下元素：

* 新的 HTML 元素： `<template>`, `<content>` 和 `<shadow>`
* 相关 API: `HTMLTemplateElement`, `HTMLContentElement` 和 `HTMLShadowElement`.
* 扩展了 `<link>` 和 `HTMLLinkElement`
