# 媒体查询（Media Queries）

从 CSS 版本2开始，就可以通过媒体类型在 CSS 中获得媒体支持。

```html
<link rel="stylesheet" href="site.css" media="screen">
<link rel="stylesheet" href="print.css" media="print">
```

在上述代码中，`media` 属性定义了不同媒体类型的样式表：

* screen 适用于计算机彩色屏幕
* print 适用于打印预览模式下查看的内容或打印内容
